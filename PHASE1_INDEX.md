## 🔧 Technical Stack

### Frontend:
- **Streamlit** - Web framework
- **HTML/CSS** - Custom styling
- **JavaScript** - (via Streamlit components)

- **Python 3.13+** - Core language
- **Ultrathink Framework** - Analysis engine
- **AWS Bedrock** - AI provider
- **SQLite** - Knowledge base

### Deployment:
- **Local** - Runs on user's machine
- **Port 8501** - Default web server
